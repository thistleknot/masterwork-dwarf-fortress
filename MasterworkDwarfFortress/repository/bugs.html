Transformations like Guilds, Groups or Mages dont work, or animals are not trained in skills.
Bug:
A reaction is run, the items used up, but a creature does not transform or does not get the wanted effect.
Solution:

Dont run it on repeat 'R'. Autosyndrome, the script behind it, does not run when the reaction is set to repeat, so just queue them up normally.

3. type "c = dfhack.gui.getSelectedUnit()" (case sensitive)
3. now "c.enemy.normal_caste = number" where number is one of the list below.
4. unpause the game, the dwarf should transform.

Boneyard uses up bones way too fast.
Bug:
Each reaction uses up one full stack, but only produces one item. One single bone makes one item, but a stack of 20 bones will also just make one item.
Solution:
No solution, this is hardcoded. The more cut-off bodyparts you have, the easier it is to get small stacks or single bones. Butchering big animals will produce big stacks, these are better used in the craftsdwarf.

 
Letters are missing, lines are black, no text.
Bug:
Parts of words are missing, letters flicker in and out of existence, long names disappear. A known Vanilla DF TrueTypeFont Bug.
Solution:
No solution, this is hardcoded. You can press F11x2 to fix it temporarily, OR press F12 to turn of TrueType. TrueType is required for the custom Font to work.

 
Workshop reactions are red, even with all the materials available.
Bug:
If a workshop is in a burrow, even if the burrow is inactive, then it accepts only materials within the same burrow.
Solution:
Make a bigger burrow including the materials, or keep the material stockpiles within the same burrow. Or delete the burrow.

 
I have many weird boulders lying around, named sparks, bacteria or smoke...
Bug:
Many things in the mod depend on temperature. Things that should melt or boil away don't without it.
Solution:
Open the GUI or Init.txt and set Temperature to ON. This can be done in a running fortress, you only have to restart DF. You do not need to gen a new world. You can also turn off "smoke effects" in the misc features section. This will decrease the amount of these boulders significantly.

 
Kitchen cant make candy or bake bread.
Bug:
Df cant find the sugar/flour inside a bag that is stored inside a barrel.
Solution:
Use the sugar/flour directly after you created it, or store it in a stockpile that is set to accept no barrels.

I have an error log.
Bug:

The Dwarf Fortress main folder shows a errorlog.txt. This is created when certain entry in the raws are missing. In 90% of the cases this is completely harmless. It is caused by enabling some settings, and disabling other.

For example you remove warpstone. The error log will tell you that warpstone is missing in the reactions for warpstone landmines and turrets.
Solution:
The default setting has no error log. If your setting has a very big error log, please leave me a message in the forum. Otherwise just ignore it, it will do no harm.
